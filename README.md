## How to Use

1. **Load a Video**: Paste any YouTube video or playlist URL in the input field
2. **Adjust View**: Use the theatre mode toggle to change video size
3. **Set Timer**: Choose between Pomodoro timer or simple counter
4. **Track Progress**: Update completed/total videos to track your course progress
5. **Stay Focused**: Minimize distractions and maximize productivity!

## Pomodoro Timer

- Set custom work duration (default: 25 minutes)
- Set custom break duration (default: 5 minutes)
- Automatic phase switching between work and break
- Visual indicators for current phase

## Course Progress

- Input completed videos and total videos
- Visual progress bar with percentage
- Collapsible section to maximize screen space
- Persistent tracking during your study session

## Tech Stack

- Pure Vanilla JavaScript (ES6+)
- Tailwind CSS (via CDN)
- No build tools required
- No external dependencies

## Local Development

Simply open `index.html` in your browser. No build process needed!

```bash
# Clone the repository
git clone https://github.com/brokenspagheti/focus-study-youtube.git

# Open in browser
open index.html
```
